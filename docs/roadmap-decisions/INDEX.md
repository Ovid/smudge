# Roadmap Decision Log Index

This index lists every /roadmap run in reverse chronological order. Each entry
captures issues found by /pushback (after the design) and /alignment (after the
plan), along with how each was resolved.

The purpose is evidence: a body of receipts that the upstream skills
(brainstorming, writing-plans) miss real things even when run by the most
capable model. Severity counts in the table below let a year of entries be
skimmed at a glance for patterns.
